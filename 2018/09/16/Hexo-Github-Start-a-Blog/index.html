使用Hexo+Github搭建博客

2018-09-16
  • 教程

前言：

大概分为以下几个步骤

  1. 搭建环境(node.js和Git环境)
  2. 安装Hexo
  3. 配置Hexo
  4. 同步到Github
  5. 发布文章

安装node.js

Hexo是基于nodeJS环境的静态博客 点击去下载

根据自己的Windows版本选择相应的安装版本

保持默认，一路Next，直到安装完成

检查node环境是否正常

快捷键Win+R，打开运行窗口

在运行窗口中输入cmd，回车，打开命令行界面
在命令行中输入

node -v
npm -v

如图所示显示版本号，安装正常

安装Git环境

下载安装Git

点击去下载

和上面一样，下载完成后打开安装包

大部分安装默认设置，只有下图需要选择第二项

检查Git环境是否正常

打开命令行，输入：

git –version

如图所示显示版本号，安装正常

安装Hexo

新建一个文件夹，用来存放博客的文件

打开命令行cd到新建的目录

cd到目录后输入

npm install -g hexo-cli

等待安装完成
